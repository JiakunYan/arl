# ARL Changelog
## [0.1.0] - 2019-03-31
### New features
- Thread management module: init, run, finalize
- AM module: rpc, rpc_agg, rpc_ff
- Collective communication: barrier, broadcast, reduce_one, reduce_all
- Data structure: HashMap, BloomFilter(local)

### New benchmarks
- Microbenchmark: bandwidth
- Kmer counting
- Contig generation
